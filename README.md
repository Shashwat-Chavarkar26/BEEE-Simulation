A Basic Electrical & Electronics Engineering (BEEE) Mini-Project <br>
An interactive, web-based simulation of an Operational Amplifier (Op-Amp) functioning as a voltage comparator. This project demonstrates the fundamental concept of Open-Loop Gain (AOL) and Saturation in analog electronics.

🚀 Overview
---
This project serves as a "Virtual Lab" for students to visualize how an Op-Amp makes binary decisions based on analog voltage inputs. Instead of static components, this simulation provides real-time graphical feedback of input/output relationships.<br>
Key Features<br>
Dual Mode Operation: Toggle between Inverting and Non-Inverting configurations.<br>
Real-Time Graphing: Dynamic Canvas-based plotting of Vin, Vref, and Vout.<br>
Visual Indicators: Virtual LED output to demonstrate the saturation state.<br>
Responsive Dashboard: Clean UI built with Vanilla JS, HTML5, and CSS3.<br>
